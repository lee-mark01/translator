# Session 001 — 2026-03-26

프로젝트 초기 논의 및 설계 문서 구조 세팅.

## Discussion
1. **프로젝트 목표 논의**: 참예수교회 실시간 설교 통역 (한→영/중)
2. **기술 조사 완료**:
   - STT: Azure Speech Services 추천 (한국어 최강, 100-300ms)
   - 번역: GPT-4o-mini + 성경 용어집 (맥락 번역 가능)
   - TTS: Google Cloud TTS (무료 티어 충분)
   - 비용: 월 $17-25 (종량제, 유연 조절 가능)
3. **오디오 입력 방식**: OBS 컴퓨터에서 가상 오디오 케이블로 연결 결정
   - 교회가 이미 OBS → YouTube 실시간 송출 중
   - 별도 장비/서버 불필요
4. **클라이언트**: 웹앱 (QR 접속, 앱 설치 불필요)
   - 끊김 대응: 자동 재연결 + 텍스트 백필
5. **호스팅**: 교회 OBS 컴퓨터 (클라우드 불필요)
   - 서버 부하 거의 없음 (API 중계 역할만)
   - 교회 인터넷 업로드 5Mbps 이상 필요 → speed test 필요
6. **통역 품질 전략**:
   - 문장 완성 후 번역 (정확도 > 속도)
   - 참예수교회 전용 용어집
   - 이전 문장 컨텍스트 유지
7. **프로젝트 관리 구조 합의**:
   - docs/ 기반 외부 기억장치
   - 세션 히스토리 기록
   - 한 번에 하나의 TASK만
   - 모호하면 질문
