# 요구사항 정의서

## 프로젝트 개요
참예수교회(True Jesus Church) 실시간 설교 통역 시스템.
목사님의 한국어 설교를 실시간으로 영어/중국어로 통역하여 외국인 참석자에게 제공.

## 교회 정보
- 교단: 참예수교회 (True Jesus Church) — 세계 각국에 분포
- 예배 일정: 화, 금, 토 — 주 3회, 각 1시간
- 현재 송출: OBS Studio → YouTube 실시간 스트리밍 중
- 외국인 참석자: 10명 이상

## 핵심 요구사항

### P0 (필수 — 1차 MVP)
1. **실시간 통역**: 한국어 설교 → 영어/중국어 음성 출력
2. **정확한 통역**: 설교 맥락 + 성경/신학 용어 정확도 우선
3. **놓치지 않는 통역**: 문장 단위 완성 후 번역 (정확도 > 속도)
4. **실시간 자막**: 번역된 텍스트를 화면에 동시 표시
5. **QR 접속**: 앱 설치 없이 QR코드 → 브라우저로 바로 접속
6. **끊김 복구**: 연결 끊겼다가 재연결 시 놓친 부분 텍스트로 복구
7. **노이즈 제거**: 목사님 목소리만 깨끗하게 입력

### P1 (2차)
8. **기록 저장**: 설교별 원문 + 번역문 타임스탬프로 저장
9. **지나간 부분 다시 보기**: 스크롤하여 이전 번역 확인
10. **번역 검증/수정**: 예배 후 틀린 번역 확인 및 수정 기능

### P2 (향후)
11. **언어 확장**: 영어/중국어 외 추가 언어
12. **다른 교회 확장**: 멀티테넌트 지원 (타 교회 요청 시)

## 기술 환경 제약
- 오디오 입력: OBS 컴퓨터의 가상 오디오 출력 활용
- 서버: 교회 OBS 컴퓨터에서 실행 (별도 클라우드 서버 없음)
- 클라이언트: 모바일 브라우저 (Wi-Fi 또는 LTE/5G)
- 비용: API 종량제 (월 $17-25 목표, 유동적으로 조절 가능)

## 비기능 요구사항
- 지연: 2-3초 이내 (사람 통역사 수준)
- 동시 접속: 최소 20명
- 가용성: 예배 시간 중 안정적 동작
- 보안: 설교 오디오/텍스트 데이터 외부 유출 방지
